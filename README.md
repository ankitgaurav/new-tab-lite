# new-tab-lite :fire:
> Upgrade the new tab page in Google Chrome 

+ Incredibly fast
+ Very lightweight (~15KB)
+ Recent bookmarks on the top bar
+ Customizable speed dials
+ Useful shortcuts in the laucher area just like Firefox
